목록 관련 태그

ul(Unordered List) 태그 : 순서가 없는 목록을 나타낼 때 쓰는 태그
ol(Ordered List) 태그 : 순서가 있는 목록을 나타낼 때 쓰는 태그
li(List item) 태그 : ul, ol 태그 내부에 들어가 목록의 요소를 나타내는 태그

ul 태그

    li 없는데에 쓰면 목록으로 인식이 되지 않음
    줄바꿈도 의미 없음

ol 태그

* ol 태그의 type 속성
type= "a" -> 소문자 알파벳 순서
type= "A" -> 대문자 알파벳 순서
type= "i" -> 소문자 로마 숫자 순서
type= "I" -> 대문자 로마 숫자 순서
type= "1" -> 아라비아 숫자 순서(기본값)
start 속성 : 시작 번호를 지정할 경우
reversed 속성 : 역순/ 속성값이 따로 없음
